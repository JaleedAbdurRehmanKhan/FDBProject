Design Highlights

Hierarchy (Specialization)

The Battery superclass explicitly maps to three child entities (LFP, NMC, LTO) using disjoint constraints.

Time-Series Tracking

The Sensor_Reading acts as a weak entity handling millions of high-frequency records, preserving 3NF rules.

Fault Anomalies

The Fault_Log table is populated exclusively via backend SQL Triggers monitoring the data streams.
